Projects Page:

  1. 1: Business Button: located on the Degree Programs Page - This button displays all the degree programs affiliated with the Business field.
  2. 2: Technology Button: located on the Degree Programs Page - This button displays all the degree programs affiliated with the Technology field.
  3. 3: In-State-Calculator: located on the Tuition Page - This calculator takes the number of semesters and outputs the in-state price.
  4. 4: Out-of-State-Calculator: located on the Tuition Page - This calculator takes the number of semesters and outputs the out-of-state price.
  5. 5: Zip Code Rate Generator: located on the Tuition Page - This input takes a zip code and determines whether the tuition rate is in or out of state.
  6. 6: Frist, Last, Email, and Phone Number fields: located on the Tuition Page - These inputs perform a validation check and provide a green or red checkmark to the user using the "focusout" event
  7. 7: Student ID Generate Button: located on the Tuition Page - This field is readonly and it will generate a 6 digit school ID number when the "generate" button is clicked.
  8. 8: Select a Program Input: located on the Tuition Page - This field will give the user a green checkmark when an option is selected.
  9. 9: Submit Button: located on the Tuition Page - This input type will run a final validation for the entire form and then provide a confirmation message to the user.
  10. Reflection: My goal here was to create a simple mockup of a college website. A challenge I faced was in my click event for my submitButton. I feel like the code is a bit repetative but it was the best I could do. The biggest challenge I had was the final validation for the entire form. I used multiple variables instead of one to check the validation. Overall it was a fun project!
