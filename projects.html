Projects Page:

  1. ALL JavaScript activity can be found on the "Application Page"
  2. 1. Function: inputValidation - this function takes a regular expression and uses it to validate the user input for the first and last name. It then returns a boolean.
  3. 2. Function: generateStudentID - this function is used for the student ID input section on the application page. The input has been disabled by applying readonly in the html but is designed for the user to click the "generate" button and it will generate a random 6 digit number.
  4. 3. Event Listener: form.addEventListener - This listener uses the focus out event for the entire form. Inside, it makes multiple calls to the inputValidation functions and plugs in the parameters needed. It then assigns the value of each function call to a variable that will later be used to validate the entire form.
  5. 4. Button Click: genButton.AddEventListener - This button is used to invoke the generateStudentID function. It then generates the value for the student ID section on the Application.
  6. 5. Change Event: degreeSelection.addEventListener - This change event includes if/else logic to determine and make sure the user has selected an option from the select menu on the application page. It then updates the boolean value for the section validation
  7. 6. Click Event/Prevent Default/Logic: submitButton - this submit button is runs the final form validation upon being clicked. It uses if statements to makes sure that all fields have been filled out and then provides the user feedback if the form was submitted successfully or not.
  8. Reflection: My goal here was to create a simple mockup of a college website. I used the application page to implement all of my javascript. I feel like there is more JS included than the 5 elements that were required for the project but it was hard to pinpoint it. For example, I attempted to make the code more clean and efficient by making the inputValidation function take multiple parameters to minimize code repetition. A challenge I faced was in my click event for my submitButton. I feel like the code is a bit repetative but it was the best I could do. The biggest challenge I had was the final validation for the entire form. I used multiple variables instead of one to check the validation. Overall it was a fun project!
